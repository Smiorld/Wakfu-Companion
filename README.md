原项目仓库：[Nexus-Hub/Wakfu-Companion](https://github.com/Nexus-Hub/Wakfu-Companion)

这是一个面向中文 `WAKFU` 玩家使用的静态工具页，基于原项目修改而来。

## 功能

- 战斗统计
- 物品追踪
- 聊天与翻译
- 会话总结
- 快捷信息

## 使用

1. 打开 `public/index.html`
2. 把 `wakfu_chat.log` 拖进页面
3. 允许浏览器读取日志文件

默认日志目录通常为：

```text
%AppData%\zaap\gamesLogs\wakfu\logs\
```

## 分发

分发给别人时，请至少保留完整的 `public/` 目录结构。

## 说明

- 这是纯前端静态项目
- 依赖浏览器的 `File System Access API` 读取本地日志
- 推荐使用 `Microsoft Edge` 或 `Google Chrome`
